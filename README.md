# Checkmates
## About The Project
Checkmates is a mobile-first application designed to simplify splitting restaurant receipts. Instead of manually calculating totals, users can scan a receipt, automatically extract items, assign them to individuals, and calculate accurate totals.

This project is also a backend engineering learning exercise, focusing on building a clean, scalable API using proper architectural principles like separation of concerns and layered design.

Core functionality:
- Capture or upload receipt images
- Extract text using OCR
- Parse receipts into structured data
- Assign items to users (frontend)
- Calculate totals (tax/tip support planned)

---

Frontend:
